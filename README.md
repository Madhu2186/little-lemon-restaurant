# 🍋 Little Lemon Restaurant Website

A modern, responsive restaurant website built using **HTML5, CSS3, and JavaScript**. This project showcases a clean user interface, responsive layout, interactive sections, and modern web design principles.

---

## 📌 Project Overview

The **Little Lemon Restaurant Website** is designed to provide customers with an engaging online experience. It includes information about the restaurant, menu items, gallery, testimonials, reservation form, contact information, and responsive navigation.

---

## ✨ Features

- 🏠 Responsive Home Page
- 🍽️ Restaurant Menu
- 👨‍🍳 About Us Section
- 🖼️ Food Gallery
- ⭐ Customer Testimonials
- 📅 Online Reservation Form
- 📍 Contact Information with Google Maps
- 📧 Newsletter Subscription
- 🌙 Dark Mode
- 📱 Mobile Responsive Navigation
- ⬆️ Back to Top Button
- 🎨 Hover Effects & Animations
- 📊 Animated Statistics
- 🔍 Smooth Scrolling

---

## 🛠️ Technologies Used

- HTML5

- JavaScript (ES6)

- Google Fonts
